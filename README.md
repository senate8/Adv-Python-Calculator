# Adv-Python-Calculator
🧮 Smart Calculator (Python CLI)
A feature-rich command-line calculator built using Python, supporting arithmetic, scientific, and advanced operations with a clean modular structure and history tracking.

🚀 Features

🔹 Basic Operations
Addition, Subtraction, Multiplication, Division
Supports full expressions (e.g. 2 + 3 * 4)

🔹 Scientific Functions
sin(x)
cos(x)
log(x)
sqrt(x)

🔹 Advanced Functions
Power calculation (x^y)
Factorial (x!)
Percentage conversion

🔹 Extra
📜 History tracking of all calculations
⚠️ Error handling for invalid inputs
🧠 Modular and object-oriented design
🛠️ Tech Stack

Language: Python

Libraries Used:

math (for scientific functions)

📦 Project Structure
calculator/
│── main.py
│── README.md

▶️ How to Run
Make sure Python is installed
Clone the repository:
git clone https://github.com/senate8/Adv-Python-Calculator.git

cd Adv-Python-Calculator

Run the program:
python main.py
